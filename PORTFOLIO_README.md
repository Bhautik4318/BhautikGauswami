# 🚀 Bhautik Gauswami - AI/ML Engineer Portfolio

A modern, responsive portfolio website showcasing expertise in Artificial Intelligence, Machine Learning, and Data Science. Built with React 19, TypeScript, and featuring stunning animations powered by Framer Motion.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Bhautik4318/BhautikGauswami.git
   cd BhautikGauswami
   ```

2. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm start
   ```
   The portfolio will run on `http://localhost:3000`
