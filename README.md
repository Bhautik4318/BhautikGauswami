# 🚀 Neural Pulse Portfolio

A modern, interactive portfolio website built with React, TypeScript, and advanced particle animations. Features a sleek black/white theme system with enhanced visual effects.

![Portfolio Preview](https://img.shields.io/badge/Live%20Demo-Portfolio-blue?style=for-the-badge)

## ✨ Features

- **🎨 Dual Theme System**: Seamless dark/light mode switching
- **🎆 Interactive Particle Background**: 150+ animated particles with mouse interaction
- **📱 Fully Responsive**: Optimized for all device sizes
- **⚡ Modern Tech Stack**: React 18, TypeScript, Tailwind CSS
- **🎭 Smooth Animations**: Framer Motion for fluid transitions
- **🎯 Clean UI/UX**: Minimalist design with professional aesthetics

## 🛠️ Tech Stack

- **Frontend**: React 18.3.1, TypeScript
- **Styling**: Tailwind CSS, Custom CSS
- **Animations**: Framer Motion, Particles.js
- **Build Tool**: Create React App
- **Icons**: Emoji-based tech icons

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/neural-pulse-portfolio.git
   cd neural-pulse-portfolio
   ```

2. **Install dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
